But

  • Aligner deux textes (actuellement limité à l’ Iliade et sa paraphrase) pour visualiser et faciliter le repérages des correspondances lexicales entre un texte-source (Iliade) et un texte-cible.
  • Repérer, grâce aux outils,
    1. les formes qui sont communes dans le texte source et dans le texte cible ;
    2. les formes du textes source qui ont été changées dans le texte cible ;
    3. les formes du texte source sans correspondance dans le texte cible ;
    4. les formes ajoutées par le texte cible (sans correspondance dans le texte source).
  • Outil d’apprentissage et d’étude du vocabulaire grec.

Description

  • Deux méthodes d’alignement sont proposées, la première, automatique, générée par un algorithme numérique (conçu par S. Zenzaro), la seconde, manuelle, établie par les éditeurs du projet (sous la supervision de B. Kindt).
  • La visualisation de l’alignement se fait par le balayage des mots.

Options d’interaction

  • Méthode d’alignement : permet de choisir entre l’alignement automatique ou manuel.
  • : efface les options sélectionnées.
  • Formes non communes : encadre les formes homériques et les formes de la paraphrase qui se correspondent syntaxiquement mais ne sont pas identiques, même dans les cas où elles appartiennent au même lemme.
  • Formes communes : encadre les formes homériques et les formes de la paraphrase qui sont strictement identiques.
  • Ajoutés dans la paraphrase : encadre les mots de la paraphrase qui n’ont pas de correspondant dans l’ Iliade .
  • Termes homériques non repris : encadre les mots du texte de l’ Iliade qui n’ont pas de correspondant dans la paraphrase.
